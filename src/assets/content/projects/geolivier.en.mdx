## geolivier.xyz : My first React website

I hesitated to indicate that this is my first website with React, but I think it can also demonstrate my ability to adopt good development patterns in a new environment.
A simple SPA, it's an opportunity for me to introduce myself, and this site is meant to be the showcase for a much larger project I recently started working on. Stay tuned!

## The feature that stands out

Not wanting to deploy a backend architecture or use a metaframework at this stage, my content is based on MDX files which allow me to manage content just like I would with Next (markdown + React components).
The core of the system relies on Vite's capabilities to automatically index my folders and transform each MDX file into an independent React module/component. This system allows dynamically generating the list of my projects while isolating each content in a separate 'chunk', loaded only when accessed to guarantee smooth navigation. This will be the subject of a first blog post!
