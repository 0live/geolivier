# geolivier.dev

Ce projet est mon portfolio personnel, construit initialement comme une SPA simple et que j'enrichis avec un système de blog statique propulsé par MDX.

## Stack technique

J'ai choisi de rester sur une stack 100% Client-Side Rendered (CSR) pour des raisons de simplicité de déploiement, en favorisant les performances via le Lazy Loading des contenus :

1.  **Vite** : Bundleur ultra-rapide.
2.  **React 19** : Rendering et UI.
3.  **Tailwind CSS 4** : Utility-first CSS.
4.  **MDX** : Markdown for the Component Era.

Le code source est disponible sur GitHub.
